# openskills

My collection of AI agent skills for OpenClaw / Claude Code.

## Structure

```
skills/
└── <skill-name>/
    └── SKILL.md
```

Each skill is a self-contained folder with a `SKILL.md` file containing YAML frontmatter (`name`, `description`) and instructions.

## Installation

```bash
# Clone to your skills directory
git clone https://github.com/huaguihai/openskills.git

# Or copy a single skill
cp -r openskills/skills/<skill-name> ~/.claude/skills/
```
